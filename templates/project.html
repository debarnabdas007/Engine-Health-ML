DATA SPECS

  • Dataset ID NASA FD001
  • Engine Type Turbofan
  • Fault Mode HPC Degradation
  • Train Units 100 Engines
Data simulates run-to-failure trajectories. Initial wear is normal; degradation accelerates over cycles.

MISSION OBJECTIVE

To engineer a robust Machine Learning pipeline capable of predicting the **Remaining Useful Life (RUL)** of aircraft engines with high precision.

Phase 1: Healthy

Optimal Operations

Phase 2: Degradation

Fault Propagation

Phase 3: Failure

System Termination

Technical Architecture

The system ingests 26-column multivariate time-series data. It utilizes an XGBoost regressor trained on feature-engineered sensor history (Rolling Means & Lag Features) to minimize RMSE.
